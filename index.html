Política de Privacidad - Conectar Seguimiento

Última actualización: mayo de 2026

Conectar Seguimiento es una aplicación destinada al seguimiento y visualización de vehículos mediante información provista por los servicios contratados por el usuario.

Información que recopilamos

La aplicación puede utilizar datos de acceso del usuario, información asociada a servicios contratados, vehículos, recorridos, posiciones GPS, eventos y detenciones.

Uso de la información

La información se utiliza únicamente para permitir el acceso seguro a la aplicación, mostrar vehículos, recorridos históricos, ubicación en mapa y datos relacionados con el servicio de seguimiento.

Ubicación

La aplicación muestra información de ubicación de vehículos registrados en la plataforma. La aplicación no utiliza la ubicación personal del dispositivo del usuario salvo que una funcionalidad futura lo requiera y sea autorizada por el usuario.

Compartición de datos

No vendemos ni compartimos datos personales con terceros para fines comerciales. Los datos pueden ser procesados por servicios técnicos necesarios para el funcionamiento de la aplicación, como servidores, API, mapas y servicios de infraestructura.

Seguridad

Aplicamos medidas razonables para proteger la información transmitida y almacenada, incluyendo autenticación de usuarios y comunicación con servicios backend.

Conservación de datos

Los datos se conservan mientras sean necesarios para prestar el servicio contratado o cumplir obligaciones operativas, técnicas o legales.
